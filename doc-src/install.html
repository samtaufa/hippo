## Installation Sequence Dependencies - git - gitpython Base System - hippo ### GIT Install GIT v1.7.1 or later for your platform from the [GIT Pages](http://git-scm.com/) ### gitpython Install [GitPython](http://gitorious.org/git-python) v0.3 or later, one method is as in the below. 
$ git clone git://github.com/gitpython-developers/gitdb.git
$ cd gitdb
$ sudo python setup.py install
$ cd ..
$ git clone git://github.com/gitpython-developers/async.git
$ cd async
$ sudo python setup.py install
$ cd ..
$ git clone git://github.com/gitpython-developers/GitPython.git
$ cd GitPython
$ sudo python setup.py install
### ### Hippo Get a copy of hippo from [github.com](https://github.com/cortesi/hippo) 
$ git clone https://github.com/cortesi/hippo.git
$ cd hippo
$ sudo python setup.py install
